## Organização de acervo bibliotecário utilizando Árvore Avl 

**no**
Classe que permite a estruturação de uma ávore. Relacione dados por meio de dois ponteiros: esquerda e direita.
Possui elementos adicionais no seu construtor por conta do tipo de ávore: AVL --> auto balanceamento

**avl:**
Classe utilizada de acordo com o google coolab da professora da disciplina. É necessário rever a forma que foi implementada para maior entendimento de como funcionam as verificação e rotações da árvore.

**livro:**
Orienta as informações necessárias para o cadastro de um livro na biblioteca.

**main:**
Possui as funções básicas para realizar o índice remissivo. Lê o texto com as funções internas do python, separa a palavra por meio de uma função interna criada e realiza a inserção das palavras por meio de função interna.
